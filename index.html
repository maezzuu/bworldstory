B WORLD 스토리보드

프로젝트 개요

: 다양한 기능이 있는 풀드롭다운 메뉴를 직접 구현하고 싶어 프로젝트를 시작하였습니다.

이 사이트는 웹사이트와 앱 사이트의 코드 구조가 다르기 때문에, 사용자 경험을 바탕으로 직접 반응형 디자인을 만들어봤습니다. 각 플랫폼에 최적화된 레이아웃을 적용하여, 보다 원활한 사용 환경을 제공하고자 했습니다. 이 과정에서 다양한 기술을 활용해 보았습니다.

기술 스택

· HTML

· CSS

· JavaScript

· Jquery

기능 구현

· 기능구현01

"GNB를 구현하기위해 addClass와 removeClass를 활용하여 마우스가 올라갈 시 전체 메뉴가 나타나는 효과를 적용했습니다. 각 GNB 항목에 대해 개별 드롭다운 메뉴를 설계하였으며, GNB에서 마우스를 떼더라도 활성화된 메뉴 아래에 파란색 선이 유지되도록 하여 현재 사용 중인 메뉴 항목을 인식할 수 있게 만들었습니다."

· 기능구현02

"키프레임을 사용하여 자동으로 슬라이드가 되는 효과를 넣었습니다. 시각적인 즐거움을 높였습니다."

· 기능구현03

"마우스를 올릴 경우 이미지 위에 가상 요소가 생성되고, 이미지가 커졌다 줄어드는 애니메이션 효과를 추가하여 시각적인 흥미를 더했습니다."

· 기능구현04

"특정 구간에 도달하면 sticky 기능을 활용하여 페이지를 스크롤할 때 고정되는 효과를 주었고, 이미지 위에 마우스를 올리면 가상 요소를 사용해 모서리 부분이 잘리는 독특한 효과를 구현했습니다."

· 기능구현05

"마우스를 올릴 때 아이콘이 위로 올라가는 애니메이션 효과를 추가해 디자인에 생동감을 더했습니다."

· 기능구현06

"마우스를 올릴 때 색상이 반전되는 효과를 주었습니다. 또한 이미지가 서서히 커지는 애니메이션을 추가하여 시각적인 흥미를 더했습니다."

· 기능구현07

"슬라이드 토글 기능을 사용하여 + 버튼 클릭 시 아래로 콘텐츠가 펼쳐지도록 구현했습니다. - 버튼 클릭 시에는 슬라이드가 다시 올라가도록 설정하여, 사용자에게 편리한 탐색 경험을 제공합니다."

개선 및 배운점

"다양한 애니메이션 활용법을 배우며, 자바스크립트의 기능을 더욱 깊이 이해하게 되었습니다. 애니메이션을 통해 사용자 경험을 향상시키는 방법을 익혔습니다. 이 과정에서 다양한 라이브러리와 기술을 접할 수 있었습니다."
