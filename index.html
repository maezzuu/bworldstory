B WORLD 스토리보드

프로젝트 개요

: 다양한 기능이 있는 풀드롭다운 메뉴를 직접 구현하고 싶어 프로젝트를 시작하였습니다.

이 사이트는 웹사이트와 앱 사이트의 코드 구조가 다르기 때문에, 사용자 경험을 바탕으로 직접 반응형 디자인을 만들어봤습니다. 각 플랫폼에 최적화된 레이아웃을 적용하여, 보다 원활한 사용 환경을 제공하고자 했습니다. 이 과정에서 다양한 기술을 활용해 보았습니다.

기술 스택

· HTML

· CSS

· JavaScript

· Jquery

기능 구현

· 기능구현01

"GNB를 구현하기위해 addClass와 removeClass를 활용하여 마우스가 올라갈 시 전체 메뉴가 나타나는 효과를 적용했습니다. 각 GNB 항목에 대해 개별 드롭다운 메뉴를 설계하였으며, GNB에서 마우스를 떼더라도 활성화된 메뉴 아래에 파란색 선이 유지되도록 하여 현재 사용 중인 메뉴 항목을 인식할 수 있게 만들었습니다."

· 기능구현02

"마우스 클릭 시 토글 기능을 활용하여 메뉴가 부드럽게 내려갔다가 올라가는 효과를 적용했습니다. 또한, 화살표가 회전하는 애니메이션을 추가하여 시각적인 디자인을 한층 더 매력적으로 만들었습니다."

· 기능구현03

"마우스 오버 시 툴팁이 나타나도록 구현하여 사용자가 보다 편리하게 사용할 수 있도록 하였습니다."

· 기능구현04

"마우스 클릭 시 드롭다운 메뉴가 열리도록 구현하여 검색창을 표시하며, 그 아래에는 dim을 활용하여 구간을 구분할 수 있도록 설계하였습니다."

· 기능구현05

"마우스 클릭 시 GNB 메뉴가 부드럽게 스크롤되도록 설계하여 사용자가 보다 쉽게 전체 메뉴를 한눈에 확인할 수 있도록 구현하였습니다."

· 기능구현06

"메인 페이지에 스와이퍼 기능을 구현하여 사이트의 주요 내용들이 효과적으로 노출되도록 하였습니다."

· 기능구현07

"마우스 hover 효과를 활용하여 마우스를 올릴 시 전체 메뉴가 자연스럽게 드러나도록 구현하였습니다."

개선 및 배운점

"툴팁, 드롭다운, 스와이퍼 등을 통하여 사용자의 편리함을 높이고, 시각적 요소를 활용하여 디자인을 더하는 방법을 배웠습니다. 또한, 반응형 구조와, 디자인의 중요성을 이해하게 되어, 다양한 기기에서도 일관된 사용자 경험을 제공하는 방법에 대해 학습했습니다.
